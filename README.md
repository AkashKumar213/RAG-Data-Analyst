# 🤖 RAG Data Analyst Agent

**Production-ready Streamlit app** that transforms natural language into data analysis + charts. Ask "line plot sales vs date" → instant visualization!

## 🚀 Features
- **FAISS RAG** - Semantic search across PDFs/Images  
- **Agentic Plotting** - "scatter plot BU vs models" → auto-generates charts
- **Excel/CSV Analysis** - Upload data → conversational analytics
- **Ollama LLM** - Local Llama3 inference (no API costs)
- **Persistent** - FAISS index + plots survive restarts

## 🧠 Dual-Mode Intelligence
- **RAG**: Answers from your PDFs/Excel data
- **LLM**: General knowledge questions  
- **Agentic**: "line plot sales vs date" → Instant charts!

## 🛠️ Tech Stack
Streamlit - FAISS - Llama3 - Docker - pandas - sentence-transformers - matplotlib

## ⚡ Quick Start
```bash
pip install -r requirements.txt
streamlit run agent_ui.py
```
